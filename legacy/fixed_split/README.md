# Legado: split fixo

Esta pasta preserva os materiais produzidos antes da mudanca para validacao
cruzada 5-fold.

Conteudo:

- `LaTeX/sbc-final/`: versao do artigo baseada em split fixo 80/10/10;
- `LaTeX/eniac-original/`: versao longa/original preservada como historico;
- `LaTeX/slide/`: apresentacao baseada nos resultados antigos;
- `results/article_materials/`: tabelas, figuras e metricas do protocolo antigo;
- `output/pdf/`: PDF de apoio gerado no fluxo antigo.

Estes arquivos nao devem ser usados como resultados confirmatorios do artigo
atual. O protocolo atual esta documentado em `../../README.md` e
`../../docs/PIPELINE_LOCAL.md`.
